# İŞ ARAMA SİTESİ

Bu sitemizde kullanışlı bir arayüz kullanılarak güncel iş ilanları, iş başvuru formu ve iş dünyası ile ilgili çeşitli istatistikler listelendi.

#ÖZELLİKLER:
-Full responsive yapıdaki sitemizde kullanıcı dostu bir arayüz oluşturuldu
-Klasik CSS ile birlikte Bootstrap kütüphanesi kullanıldı
-Renk geçişleri hoover efektleri ile desteklendi
